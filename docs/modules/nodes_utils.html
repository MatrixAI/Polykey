calculateBucketIndex

  • calculateBucketIndex(sourceNode: NodeId, targetNode: NodeId, nodeIdBits?: number): number
    • Defined in src/nodes/utils.ts:37

    Find the correct index of the k-bucket to add a new node to. A node's k-buckets are organised such that for the ith k-bucket where 0 <= i < nodeIdBits, the contacts in this ith bucket are known to adhere to the following inequality: 2^i <= distance (from current node) < 2^(i+1)

    NOTE: because XOR is a commutative operation (i.e. a XOR b = b XOR a), the order of the passed parameters is actually irrelevant. These variables are purely named for communicating function purpose.

    Parameters

    • sourceNode: NodeId
    • targetNode: NodeId
    • nodeIdBits: number = 256

    Returns number
